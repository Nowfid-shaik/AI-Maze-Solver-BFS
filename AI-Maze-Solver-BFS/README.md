# AI Maze Solver using BFS

This project demonstrates how Artificial Intelligence can be used to find the shortest path in a maze using the Breadth First Search (BFS) algorithm.

## Features

- Maze visualization
- BFS shortest path finding
- Node exploration
- Path highlighting
- Random maze generation

## Algorithm Used

Breadth First Search (BFS)

BFS explores nodes level by level and guarantees the shortest path in an unweighted maze.

## Applications

- Robot navigation
- Game pathfinding
- Autonomous vehicles
- GPS routing

## How to Run

1. Install Python
2. Run:

python maze_solver.py
